# Advanced OS Assignment

## Overview
This repository contains three scripts developed for the Advanced Operating Systems assessment:

1. `task1_system_monitor.sh`  
   Bash-based process, disk, and log management tool.

2. `task2_scheduler.py`  
   Python-based HPC job scheduler implementing Round Robin and Priority scheduling.

3. `task3_secure_submission.sh` + `task3_login_monitor.py`  
   Bash and Python secure assignment submission and login monitoring system.

---

## Requirements
- Linux / POSIX-compliant environment
- Bash
- Python 3
